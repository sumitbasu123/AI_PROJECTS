# AI Projects

Practical AI and data applications focused on transparent automation, privacy, and human-in-the-loop decisions.

## Projects

### [Career Command Center](career-command-center)

A Streamlit job-intelligence and interview-preparation application with multi-source discovery, normalized job data, duplicate detection, credibility analysis, explainable fit scoring, pipeline tracking, and optional AI coaching.

### [AI Study](ai-study)

An offline-first, textbook-grounded learning assistant using OCR, corrective retrieval-augmented generation, LangGraph tutor and exam agents, local embeddings, and explicit hallucination guards.

### [Vibe Trading Portfolio Dashboard](vibe-trading)

An Indian-equities portfolio and paper-trading extension built on the MIT-licensed HKUDS Vibe-Trading platform, with local holdings analysis, concentration indicators, simulated orders, and guarded paper-broker connectors.

Public versions contain source code and safe configuration examples only. Credentials, personal profiles, learning material, vector databases, financial holdings, broker state, and runtime data are intentionally excluded.
