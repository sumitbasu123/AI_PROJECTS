# Security Policy

Do not publish API keys, broker credentials, account identifiers, holdings workbooks, order ledgers, or runtime logs.

For vulnerabilities in the upstream platform, use the [HKUDS Vibe-Trading private advisory channel](https://github.com/HKUDS/Vibe-Trading/security/advisories/new). For issues specific to this portfolio extension, contact this repository's owner privately through their GitHub profile. Do not place sensitive reproduction data in a public issue.
